# Silver Centauri - Mutanten Datenbank
Dieses Web-Projekt steht in Verbindung mit der Science-Fiction-Buchreihe *Silver Centauri* von Anja Dannenberg.

Die Webseite *index.html* enthält eine tabellarische Übersicht aller Mutantenprodukte des Forschungsinstituts Silver Centauri.
Zusätzlich ermöglicht ein Formular die Eingabe einer Mutantennummer. Das Skript überprüft anhand der Tabellenwerte, ob die eingegebene Nummer valide ist.

## Installation
1. Repository klonen oder herunterladen:
   ```sh
   git clone https://github.com/AnjaDannbg/SilverCentauri_MutantenDatenbank
2. Öffne die Datei index.html im Browser.

## Bedienung
### Formularfeld *Filter*
Gebe einen Wert in das Formularfeld *Filter* ein, um die Tabelle danach zu filtern. Die Suche ist case-sensitive.

### Formular *Mutantennummer*
Fülle die Formularfelder im Formular *Mutantennummer* aus, um zu prüfen, ob die Werte eine gültige Mutantennummer ergeben.

Falls die Mutantennummer valide ist, wird der passende Eintrag in der Tabelle hellrot markiert und es erscheint der **grüne Text**:
> XXXX-X X XXXXXXXX X ist eine gültige Mutantennummer.

Andernfalls werden die Felder mit falscher Eingabe rot markiert und es erscheint der **rote Text**:
> XXXX-X X XXXXXXXX X ist keine gültige Mutantennummer.

### Auswahl *Mögliche Mutantennummern*
Clicke auf den '+'-Button neben einer möglichen Mutantennummer, um seine Werte in das Formular *Mutantennummer* zu übertragen.
